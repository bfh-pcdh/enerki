# enerKI
[See it in action](https://enerki-2ddc65.pages.ti.bfh.ch).

Use Chrome for WebUSB support if you want to use ANT+ sensors.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
